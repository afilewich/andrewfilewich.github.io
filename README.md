# Andrew Filewich Academic Portfolio

This repository contains my GitHub Pages portfolio website for KNES 381 / public portfolio development.

## Project Focus

This website is centered on my honours thesis:

**Pre-Laboratory Instructional Videos in Experiential Learning: A Kinesiology Perspective**

The portfolio presents the thesis in a web-based format and highlights:

- the research problem
- the study design
- key findings
- implications for kinesiology education

## Website Goals

The goal of this site is to communicate my research clearly while demonstrating basic web development, GitHub workflow, and digital project design.
